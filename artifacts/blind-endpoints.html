A place to record the phenomenon where systems respond—but not with the truth you can see while authenticated. The grotto is not a villain; it is a warning.

Grotto

The grotto is where you learn to distrust “success.” A 200 status code can still be a lie. A build queue can still lead nowhere. A public page can still be invisible to the public.

When you record a blind endpoint, record the exact URL, the exact response, and the exact contrast with authenticated behavior.

Shadows

Shadows are partial failures: a CDN that serves JS but blocks HTML; a raw file that ships with nosniff and the wrong MIME; a Pages site that stays 404 even after configuration changes.

Shadows are where explorers get stranded—unless you leave handrails.
